# WOMProject - Technical Assessment

A React Native mobile application built as part of the Frontend & Mobile Developer technical assessment. This project demonstrates core competencies in software architecture, form validation, and efficient development workflows.

## 🚀 Tech Stack

- **Framework:** React Native (CLI)
- **Runtime & Package Manager:** [Bun](https://bun.sh/) (for ultra-fast builds and dependency management)
- **Validation:** [Yup](https://github.com/jquense/yup)
- **State Management:** React Context API / Hooks
- **Testing:** [Jest](https://jestjs.io/)
- **SDK & Build:** JDK 17, NDK 27.x, Gradle 9.0

## 🛠️ Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:
- **Node.js:** v24.14.1 or later (Managed via NVM)
- **Bun:** v1.3.11 or later
- **Java Development Kit:** Zulu OpenJDK 17
- **Android Studio:** With SDK Platform 37 and NDK 27.x configured
